# Radial Gradient Generator

A generator for radial gradients makes it easier to create box shadows for web development interactively and visually.     

Click here to visit the site: [https://jonathancklee1.github.io/learn-flags/](https://jonathancklee1.github.io/shadow-generator/)

## Features

-   **Customise gradient with sliders and inputs**
-   **Option for feTurbulence filter to generate noise on gradient**
-   **Add and remove multiple colours**
-   **Copy to clipboard for an easy "to CSS" way**

## TechStack and Packages used

-   React
-   Tailwind CSS
-   DaisyUi
-   Zustand
